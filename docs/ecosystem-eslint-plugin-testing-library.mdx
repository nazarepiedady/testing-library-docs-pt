[`eslint-plugin-testing-library`][gh] é uma extensão de ESLint para a `Testing Library` que ajuda os utilizadores a seguirem boas práticas e a antecipar erros comuns quando escrevemos testes.

![exemplo da `eslint-plugin-testing-library` exibindo um erro](/img/eslint-plugin-screenshot.png)

Esta extensão inclui:

- Várias regras para analisar o código específico da `Testing Library`.
- Configuração partilhável para regras recomendadas.
- Configurações partilháveis para regras especificadas pela abstração: Angular, React, Vue.
- Algumas regras auto-corrigíveis.

Encontre mais informações regras disponíveis e configurações na GitHub-

- [eslint-plugin-testing-library on GitHub][gh]

[gh]: https://github.com/testing-library/eslint-plugin-testing-library
