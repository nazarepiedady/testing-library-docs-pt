A [`query-extensions`][gh] é uma biblioteca acompanhante experimental para Testing Library que mistura a API de alto nível com as consultas principais da `@testing-library` padrão:

```bash npm2yarn
npm install --save-dev query-extensions
```

Consulte o [repositório da GitHub] pela documentação para exemplos e fundamento lógico. 

[gh]: https://github.com/tjefferson08/query-extensions
