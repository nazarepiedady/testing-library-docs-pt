---

A [`eslint-plugin-jest-dom`][gh] é uma extensão de ESLint para a Jest DOM que ajuda os utilizadores a seguirem as boas práticas e anteciparem-se a erros comuns quando estiverem a escrever os testes.

Esta extensão inclui:

- Várias regras para analise de código específico de Jest DOM.
- Configuração partilhável para as regras recomendadas.
- Algumas regras auto-corrigíveis.

Encontre mais informações sobre as regras e configurações disponíveis na GitHub.

- [eslint-plugin-jest-dom na GitHub][gh]

[gh]: https://github.com/testing-library/eslint-plugin-jest-dom
