A [`bs-jest-dom`][gh] é uma biblioteca acompanhante para [`bs-react-testing-library`](bs-react-testing-library/intro.mdx) que fornece correspondentes de elemento de DOM personalizados para Jest na [ReasonML][re] através da [BuckleScript][bucklescript]:

```bash npm2yarn
npm install --save-dev bs-jest-dom
```

- [bs-jest-dom na GitHub][gh]

Consulte a [documentação da `jest-dom`](https://github.com/testing-library/jest-dom) para uma lista completa dos correspondentes disponíveis.

[gh]: https://github.com/wyze/bs-jest-dom

## Configuração

```json
{
  "bs-dev-dependencies": ["bs-jest-dom"]
}
```

## Exemplo

```reason title="A_test.re"
open Jest;
open JestDom;
open ReactTestingLibrary;

module Heading = {
  let component = ReasonReact.statelessComponent("Heading");

  let make = (~text, _children) => {
    ...component,
    render: _self =>
      <h1> {ReasonReact.string(text)} </h1>,
  };
};

test("renders with text", () =>
  <Heading text="Hello, World!" />
  |> render
  |> getByText(~matcher=`Str("Hello, World!"))
  |> expect
  |> toBeInTheDocument
);
```
