A [`Jest Native`](https://github.com/testing-library/jest-native) é uma biblioteca acompanhante para `React Native Testing Library` que fornece correspondes de elemento personalizados para Jest:

```bash npm2yarn
npm install --save-dev @testing-library/jest-native
```

```jsx
const {queryByTestId} = render(
  <View>
    <View testID="not-empty">
      <Text testID="empty" />
    </View>
    <Text testID="visible">Visible Example</Text>
  </View>,
)

expect(queryByTestId('not-empty')).not.toBeEmpty()
```

> Nota: quando estiveres a usar alguns destes correspondentes, talvez precises de certificar-te de usar uma função de consulta (como `queryByTestId`) ao invés da função de recuperação (como `getByTestId`). De outro modo a função `get*` poderia lançar um erro antes da tua afirmação.

Consulte a [documentação da Jest Native](https://github.com/testing-library/jest-native) para teres acesso a uma lista completa de correspondentes disponíveis.
